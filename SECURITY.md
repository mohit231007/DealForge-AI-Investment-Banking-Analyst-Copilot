# Security and Data Handling

## Public portfolio edition

Only synthetic sample inputs should be committed to this repository.

Do not commit:

- confidential company documents;
- customer data rooms;
- API keys or tokens;
- licensed market-data exports;
- personal data;
- private product configuration;
- generated outputs containing non-public information.

## Running locally

Uploaded files in the Streamlit demo are written only to a temporary local workspace for the active run. Review and delete generated files before sharing them.

## Reporting a security issue

Do not post secrets or sensitive data in a public issue. Contact the repository owner privately through the contact method listed on the GitHub profile.
